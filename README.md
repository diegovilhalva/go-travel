# GoTravel 

**GoTravel** is a responsive landing page that allows users to discover and book fully managed, all-inclusive vacation packages at over 14,000 resorts around the globe. Built with modern web technologies, it offers a sleek interface and a lead form where users can submit their name and email to stay informed.
