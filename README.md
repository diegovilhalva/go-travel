# GoTravel 

**GoTravel** is a responsive landing page that allows users to discover and book fully managed, all-inclusive vacation packages at over 14,000 resorts around the globe. Built with modern web technologies, it offers a sleek interface and a lead form where users can submit their name and email to stay informed.

🔗 [Live Site](https://gotravel-dr.netlify.app/)  
📁 [GitHub Repository](https://github.com/diegovilhalva/go-travel)

---

## ✨ Features

- 🖥️ Fully responsive landing page
- 📄 Blog posts and sections loaded dynamically from Supabase
- 📬 Lead form to collect user name and email
- ⚡ Smooth animations using Framer Motion
- ☁️ Integrated with Supabase for backend and media storage
- ⚙️ Deployed on Netlify with Google Analytics enabled

## 🛠️ Setup

1. Clone the repository:

```bash
git clone https://github.com/diegovilhalva/go-travel
cd go-travel
````

2. Install dependencies:

```bash
npm install
```

3. Create a `.env` file in the root and add the following variables:

```
VITE_SUPABASE_API_KEY=your-api-key
VITE_MEASUREMENT_ID=your-google-analytics-measurement-id
```

4. Run the development server:

```bash
npm run dev
```
